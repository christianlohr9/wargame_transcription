     ```
     Notify.create({

       type: 'positive',
       position: 'top-right',
       timeout: 5000,
       actions: [{ icon: 'close', color: 'white' }]
     })
     ```

2. Ensure the notification fires only when summary/prompt results arrive, not for other SSE events. Use the existing `data.type === 'prompt'` check that the emitter already uses.
